## [0.1.0] - 2026-01-12

### Added

- Initial public release
- MongoDB-native in-memory projection layer
- Change Streams synchronization
- Await semantics for deterministic read-after-write consistency
- Four index types: Inverse, Inverse Unique, Sorted, Suffix
- Event listeners: Before, After, Notify/AwaitNotify
- Comprehensive documentation
- Examples: CRUD, indexes, listeners
- GitHub hygiene files (CONTRIBUTING.md, SECURITY.md, etc.)
- CI/CD workflow
